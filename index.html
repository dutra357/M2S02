Sobre:

Seja bem-vindo ao meu site pessoal! Aqui você poderá encontrar informações sobre o meu trabalho, stacks e tecnologias as quais possuo experiência profissional, bem como um portfólio selecionado de trabalhos anteriores já entregues.

Graduado em Direito, advoguei por mais de 12 anos na seara cível e tributária, tendo hoje migrado de carreira para a área de desenvolvimento de software. Ainda cursando engenharia de software, tenho ainda formação de desenvolvimento WEB-FullStack pelo SENAI/SC.

Tendo interesse em nos contatar, peço gentilmente que utilize o formulário abaixo, ou mesmo nos encaminhe mensagem privada em alguma de nossas redes.

Stacks:

Trabalho com desenvolvimento back-end em Java (Spring Boot), JavaScript (Node) e bancos de dados relacionais PostgreSQL. Ainda que não seja meu principal foco, desenvolvo front-end com as tecnologias básicas de HTML, CSS e linguagem JavaScript. Vale assinalar ainda alguma experiência dom Python e PHP.
